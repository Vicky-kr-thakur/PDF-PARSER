API documentation
=================

Functions and classes a user of `pdf4py` can use in its projects are collected into a bunch
of modules listed below. The most important one is :ref:`parser_module` which defines the class
`Parser`, a (almost) conforming PDF parser whose coverage of PDF syntax rules is illustrated into
:ref:`standard_coverage`.
